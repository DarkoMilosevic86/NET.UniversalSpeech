Welcome to the NET.UniversalSpeech

 The NET.UniversalSpeech is the parte of the Quentin C's UniversalSpeech which contains the various functions exported from the UniversalSpeech.dll file.
This Microsoft .NET assembly contains ANSI and Unicode version of functions.
